    Git commands(base):
  • git config --global user.name blabla - Глобальний конфіг(налаштування) для локальних репозиторіїв з ім'ям користувача
  • git config --global user.email blabal@bla - Глобальний конфіг(налаштування) для локальних репозиторіїв з електронною поштою
  • git init - Команда ініціалізує локальний git репозиторій
  • git status - Стан файлів в робочій директорії і індексі: які файли змінені, але не додано в індекс; які очікують коммітів в індексі.
  • git add - Додавання вмісту робочої директорії в індекс (staging area) для подальших коммітів.
  • git commit - Зберігаггя індексованих змін з коментарем
  • git log - Перегляд історії комітів
  • git checkout - Переключення гілок або відновлення файлів робочого дерева
  • git branch - Список, створення або видалення гілок
  • git push - Завантаження змін з локального репозиторію на віддалений
  • git pull - Завантаження змін з віддаленого репозиторія на локальний
  • git merge - Об'єднання кількох гілок в одну
  • git clone - Копіювання віддаленого репозиторію(в іншудиректорію наприклад)

.gitignore це файл в який вносяться назви файлів і папок які необхідно приховати від git

    Git commands(advanced):
  • git revert - Повернення деяких існуючих комітів
  • git reset - Інструмент для скасування змін
  • git rebase - Перезапис історії коммітів при злитті гілок
  • git cherry-pick - Перенесення комітів між гілками без повного злиття
  • git stash - Зберігання змін у фалайх в тим часове сховище, з можливістю повернення до них
